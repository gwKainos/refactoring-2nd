# refactoring-2nd

```shell
# package.json 파일 생성
$ npm init -y

# typescript 패키지 추가 (devDependencies)
$ npm i -D typescript

# 타입스크립트 실행 도구 설치 (devDependencies)
$ npm i -D ts-node

# 서버 자동 재시작 툴
$ npm install nodemon

# tsconfig.json 생성
npx tsc --init

# jest, ts-jest 패키지 설치
npm install --save-dev jest ts-jest @types/jest

# jest.config.js 설정 파일 생성
npx ts-jest config:init
```
